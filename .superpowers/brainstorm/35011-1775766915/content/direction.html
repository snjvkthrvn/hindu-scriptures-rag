📚 The Scholar's Workbench

For the researcher. Browse sidebar (filter by scripture, school, language), slash commands (/compare, /deep, /story), verse lookup (paste BG 2.47 to see it directly), school-by-school comparison views, structured output with metadata. Optimizes for finding and cross-referencing.

🪔 The Sacred Interface

For the reverent reader. Dark mode with candlelight palette, dramatic Devanagari typography, subtle mandala/yantra watermarks, gentle verse-reveal animations, distinctive per-scripture visual identities, PWA install for daily reading. Optimizes for feeling the weight of the text.

My recommendation

Reading Companion + Sacred Interface. They reinforce each other — the persistence/actions make the app worth returning to, and the typography/dark mode make each return feel meaningful. Scholar's Workbench is powerful but adds complexity (new routes, sidebar navigation) that might dilute the meditative feel. Could be a v2 direction.
