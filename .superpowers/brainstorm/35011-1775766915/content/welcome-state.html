How should the empty/welcome state feel?

Three takes on the first screen. All remove the current descriptive paragraph ("I search across 190,000+ verses...") — that's chrome. The question is what replaces it. Shown in dark Temple Ember since that's where the mood lives.

A · Verse of the Day

A single verse greets you, rotating daily (or randomly from a curated list). The Om sits above. Below, one soft line of invitation. No prompt chips at all. Users type their first question — but if they're stuck, they can ask about the verse shown.

Feels like: opening a book of daily readings. Most contemplative, gives the user a reason to come back each day.

B · Gentle Prompts ★

Large Om, title, Sanskrit subtitle. Then prompts rendered as italic serif lines separated by thin rules — not pill buttons. They read like invitations written in the margins of a book, not UI controls. Click any to begin.

Feels like: a thoughtful contents page. Quiet, still functional for first-time users who need a starting point.

C · Pure Opening

Just the Om, a gold rule, and one line. No prompts, no description, no tagline. The input bar below carries the weight of what to do next. Maximum reverence and minimum scaffolding.

Feels like: a temple doorway. Risk: new users without a question in mind will bounce — there's nothing to click.
My recommendation

B · Gentle Prompts — honors the "less chrome, more quiet" principle you chose for the drawer while still giving newcomers a way in. The italic-serif prompt style is a huge aesthetic upgrade over the current saffron pill chips, and it matches the literary voice of the LLM output. Option A (Verse of the Day) is beautiful and I'd include it as an additional feature once the core ships — but as the only welcome content, it makes the product feel like a daily-reading widget rather than a Q&A tool, which isn't what it is. Option C is gorgeous but leaves first-time users at sea.

Hybrid worth considering: Start with B (Gentle Prompts). Once user has used the app a few times, the welcome state could optionally show a "Verse of the Day" above the prompts. I can note this as a v2 feature if you like.
